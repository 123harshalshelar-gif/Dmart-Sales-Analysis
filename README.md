# 🛒 Dmart Sales Analysis Project

##  Overview

This project analyzes Dmart sales data using Excel to uncover business insights such as sales trends, customer behavior, and payment preferences.

##  Tools Used

* Microsoft Excel
* Pivot Tables
* Charts (Pie Chart, Bar Chart)

##  Dataset Features

* Store ID, City, State
* Product Category & Name
* Quantity & Price
* Discount Applied
* Total Amount
* Payment Method

##  Key Insights

* Identified top-performing cities and stores
* Analyzed impact of discounts on total sales
* Found most used payment methods
* Compared product category performance

##  Visualizations

* Pie chart for payment methods
* Bar chart for city-wise sales
* Sales distribution analysis

##  Conclusion

The analysis helps understand sales patterns and customer preferences, which can support better business decisions.
